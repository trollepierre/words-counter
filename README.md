# Words Counter

> A library to count words' occurrence   

## Init environment for development

``` bash
# install Yarn
npm install -g yarn@1.10.1
```

## Build Setup

``` bash
# install dependencies
yarn
```

## Lint and Test
```

# start linter
yarn lint

# fix linter errors
yarn lint:fix

# run all tests
yarn test

# run all tests in watch mode
yarn test:watch
```
